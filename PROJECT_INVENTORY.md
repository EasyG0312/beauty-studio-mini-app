## 🔧 ЧТО НУЖНО ДОБАВИТЬ / ДОРАБОТАТЬ

### ✅ СДЕЛАНО СЕГОДНЯ:

1. **✅ Умная система записи** — фильтрация по занятости
   - При выборе мастера сначала → показываются только его свободные слоты
   - При выборе даты/времени сначала → показываются только свободные мастера
   - API `/api/masters/available?date=&time=` — мастера со свободным слотом
   - API `/api/slots/{date}?master=` — слоты для конкретного мастера

2. **✅ Real-time проверка слота** — защита от двойной записи
   - При нажатии "Записаться" — проверка API перед сохранением
   - Если слот занят — ошибка и автоматический возврат к выбору времени
   - Обновление списка доступных слотов в реальном времени

3. **✅ API для работы с клиентской базой**
   - `GET /api/clients/inactive?days=60` — "забытые" клиенты (>2 месяцев)
   - `GET /api/clients/{id}/ltv` — Lifetime Value клиента
   - Расчёт: всего потрачено, средний чек, количество визитов

4. **✅ Telegram уведомления админу** (интеграция в Mini App бот)
   - Уведомление о новой записи с кнопкой просмотра
   - Уведомление об отмене записи
   - Уведомление о новом отзыве с оценкой
   - Кнопка быстрого доступа к Mini App

5. **✅ Отзывы клиента** — клиент оставляет отзыв после посещения
   - Страница `/create-review?booking={id}` — интерактивные звёзды (1-5)
   - Комментарий необязательный
   - После отправки — уведомление админу в Telegram
   - Кнопка "Оценить" в истории посещений

6. **✅ История посещений клиента** — детальная история с суммами
   - Страница `/visits` — список всех визитов
   - Показывает: дату, услугу, мастера, сумму, статус
   - Статистика: всего визитов, потрачено
   - Кнопка "Оценить" для выполненных без отзыва
   - Фильтры: Все / Выполнены / Отмены

7. **✅ Загрузка фото в портфолио** — для мастеров
   - Страница `/portfolio/upload` — drag & drop загрузка
   - Выбор категории (маникюр, стрижка, окрашивание...)
   - Превью фото перед публикацией
   - Поле для описания работы
   - Backend POST /api/portfolio с FormData
