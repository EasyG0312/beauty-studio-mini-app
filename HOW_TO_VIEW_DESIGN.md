## 3️⃣ **TELEGRAM WEBAPP (Как будет у пользователей)**

### Через @BotFather:

1. Откройте @BotFather в Telegram
2. Выберите вашего бота
3. Bot Settings → Menu Button
4. Укажите URL вашего приложения

**Для локальной разработки (ngrok):**
```bash
# Terminal 3: ngrok туннель
ngrok http 5173
```

Получите URL типа: `https://abc123.ngrok.io`

Укажите этот URL в @BotFather.
