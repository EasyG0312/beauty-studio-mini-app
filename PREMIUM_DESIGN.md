## 🎯 КЛЮЧЕВЫЕ ИЗМЕНЕНИЯ

### 1. Цветовая палитра

**Было (Pink):**
```css
--brand-primary: #E91E8C  /* Pink */
```

**Стало (Gold & Emerald):**
```css
--brand-gold: #D4AF37      /* Золото */
--brand-emerald: #50C878   /* Изумруд */
--brand-powder: #F5F0EB    /* Пудровый */
--brand-beige: #F9F7F5     /* Бежевый */
```

### 2. Glassmorphism

**Эффект матового стекла:**
```css
.glass {
  background: rgba(255, 255, 255, 0.72);
  backdrop-filter: blur(20px) saturate(180%);
  border: 1px solid rgba(255, 255, 255, 0.5);
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.08);
}
```

**Применение:**
- Навигационная панель
- Модальные окна
- Плавающие элементы

### 3. Скругления (Large Radius)

**Было:** 12-16px
**Стало:** 16-32px

```css
--radius-lg: 20px;   /* Карточки */
--radius-xl: 24px;   /* Кнопки */
--radius-2xl: 32px;  /* Модальные окна */
```

### 4. Тени (Apple-style)

**Мягкие, рассеянные:**
```css
--shadow-lg: 0 8px 32px rgba(0, 0, 0, 0.12);
--shadow-gold: 0 8px 32px rgba(212, 175, 55, 0.3);
```

### 5. Анимации (Spring Physics)

**Пружинистый отклик:**
```css
--transition-spring: 500ms cubic-bezier(0.175, 0.885, 0.32, 1.275);
```

```css

    --tg-theme-bg-color: #000000;

  }
}
```
