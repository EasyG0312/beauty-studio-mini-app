## 🏗️ АРХИТЕКТУРА ИЗМЕНЕНИЙ

### Backend (новые эндпоинты)
```
GET    /api/profile                    # Профиль клиента
PUT    /api/profile                    # Обновить профиль
GET    /api/profile/bonuses            # Бонусный баланс
POST   /api/bookings/{id}/qr           # Генерация QR-кода
GET    /api/reviews/{id}/photo         # Фото отзыва
POST   /api/reviews/{id}/photo         # Загрузка фото
GET    /api/certificates               # Мои сертификаты
POST   /api/certificates               # Создать сертификат
GET    /api/visits                     # История посещений
```

### Frontend (новые страницы)
```
/profile          # Страница профиля
/profile/edit     # Редактирование
/visits           # История посещений
/certificates     # Подарочные сертификаты
```

### База данных (новые таблицы/колонки)
```
clients:
  + bonus_balance INT
  + total_spent INT

reviews:
  + photo_url TEXT

certificates (новая таблица):
  - id
  - code
  - amount
  - created_by
  - used_by
  - used_at
  - is_active

qr_codes (новая таблица):
  - id
  - booking_id
  - code
  - created_at
  - scanned_at
```
